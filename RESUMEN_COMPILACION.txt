COMO DISTRIBUIR EL PROGRAMA:
=============================

OPCION 1: Entregar la carpeta completa
---------------------------------------
1. Copia toda la carpeta "Distribucion_GlosasAutomatizador"
2. Enviala al usuario por correo, USB o red compartida
3. El usuario solo debe hacer doble clic en GlosasAutomatizador.exe


OPCION 2: Comprimir en ZIP
---------------------------
1. Haz clic derecho en "Distribucion_GlosasAutomatizador"
2. Selecciona "Enviar a > Carpeta comprimida"
3. Envia el archivo ZIP al usuario
4. El usuario debe extraer todo y ejecutar el .exe


IMPORTANTE PARA LA DISTRIBUCION:
=================================

✓ SIEMPRE incluir credentials.json junto al ejecutable
✓ Ambos archivos deben estar en la MISMA carpeta
✓ El usuario NO necesita instalar Python
✓ El usuario NO necesita instalar dependencias
✓ Funciona en cualquier PC con Windows 7 o superior
✓ El programa es portable (no requiere instalacion)


CARACTERISTICAS DEL EJECUTABLE:
================================

- Tamaño: 28 MB
- Incluye todas las dependencias necesarias
- No requiere instalacion
- No deja archivos en el sistema (excepto en AppData)
- Ventanas graficas (no requiere linea de comandos)
- Logs automaticos para depuracion


DONDE GUARDA DATOS EL PROGRAMA:
================================

El programa guarda sus datos en:
C:\Users\[USUARIO]\AppData\Local\GlosasAutomatizador\

Archivos que crea:
- token.json: Token de autenticacion de Google (se crea al primer uso)
- processed_ids.json: IDs de correos ya procesados
- glosas_downloader.log: Archivo de log con historial
- downloads/: Carpeta temporal (se borra despues de crear el ZIP)

El ZIP final se guarda en el ESCRITORIO del usuario.


RECOMENDACIONES:
================

1. PRUEBA EL EJECUTABLE en tu PC antes de distribuirlo:
   - Ve a la carpeta Distribucion_GlosasAutomatizador
   - Haz doble clic en GlosasAutomatizador.exe
   - Verifica que funcione correctamente

2. Si necesitas recompilar en el futuro:
   - Ejecuta build_exe.bat
   - O usa: pyinstaller --clean GlosasAutomatizador.spec

3. IMPORTANTE: Si actualizas credentials.json:
   - Debes volver a distribuir el programa completo
   - Los usuarios deben volver a autenticarse


SEGURIDAD:
==========

El programa:
- Solo tiene permisos de LECTURA de Gmail
- NO puede enviar correos
- NO puede borrar correos
- NO puede modificar correos
- NO envia datos a ningun servidor externo
- Solo se comunica con Google Gmail API


SOPORTE TECNICO:
================

Si los usuarios tienen problemas:
1. Revisar el archivo glosas_downloader.log en AppData
2. Verificar que credentials.json este presente
3. Verificar conexion a internet
4. Verificar permisos de Gmail
